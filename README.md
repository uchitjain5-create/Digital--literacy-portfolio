# Digital--literacy-portfolio
This project demonstrates essential digital literacy skills required for students in today’s digital environment. It covers key areas such as creating a professional online presence, using digital tools, effective communication, collaboration, and cyber safety.
Name: Uchit Jain Registration Number: 25BAI10798 Course: CSE0001 – Digital Literacy University: VIT Bhopal University

📌 Project Overview This project demonstrates essential digital literacy skills required for students in today’s digital environment. It covers key areas such as creating a professional online presence, using digital tools, effective communication, collaboration, and cyber safety.

🧩 Tasks Included 🔹 Task 1: Infographic Created a digital literacy infographic using Canva, covering topics such as definition, importance, safe practices, useful tools, and email etiquette.

🔹 Task 2: Digital Portfolio Developed professional profiles on GitHub, LinkedIn, and Kaggle to establish a strong digital presence and portfolio.

🔹 Task 3: Platforms Completed a coding challenge on HackerRank and created a Google Form quiz to understand online collaboration and assessment tools.

🔹 Task 4: Email Etiquette Drafted professional emails and created a social media do’s and don’ts checklist to understand responsible communication.

🔹 Task 5: Cybercrime Awareness Studied phishing as a cybercrime and created a prevention checklist to promote safe online practices.
